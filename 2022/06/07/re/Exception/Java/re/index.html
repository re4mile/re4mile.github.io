throw | throws

总结：

throw和throws的区别：

（1）位置不同：

throw：在函数内部去throw

throws: 在函数的头部去throws

（2）内容不同：

throw 一个异常的对象

throws 一个异常的类

（3）作用不同：

throw ：异常发生的地方

throws : 函数的调用者
