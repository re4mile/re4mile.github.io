ctype.h

isprint 检测是否可打印

空格也可以打印

int isprint(int ASCII)

可打印返回非零

否者返回0

输出

isgraph 检测是否有图形表示(空格除外)

int isgraph(int ASCII)

可打印返回非零(空格除外)

否者返回0

isalnum 检测是否是数字或者字母

int isalnum(int ASCII)

检测你的ASCII是不是数字或者字母

1不是ASCII的1

‘1’才是ASCII的‘1’

函数

返回非零状态,表示字母数字

返回零,不是字母数字

输出

isalpha 检测是不是字母

int isalpha(int ASCII)

是字母返回非零

否者返回0

输出

islower 检测是小写字母

int islower(int ASCII)

是小写字母返回非零

否者返回0

输出

isupper 检测是大写字母

int isupper(int c)

是大写字母返回非零

否者返回0

输出

isdigit 检测是不是10以内的ASCII->10进制数字

int isdigit(inr ASCII)

是10进制数字返回非零

否者返回0

‘0x20’不是10进制数字,更像一个字符串

输出

isxdigit 检测是不是16进制的字符串

是16进制数字返回非零

否者返回0

输出

iscntrl 检测是不是控制字符

int iscntrl(int ASCII)

是控制字符返回非零

否者返回0

控制字符不可打印

输出它的10进制

ispunct 检测是不是标点符号

int ispunct(int ascii)

可标点符号返回非零(空格除外)

否者返回0

输出

isspace 检测是不是空白字符

int isspace(int c)

空白字符:

是返回非零

否者返回0

输出

tolower 大写变小写

int tolower(int 大写)

返回值是小写的值

如果原本本来就是小写,或者他不是字母就不处理

输出

toupper 小写变大写

int toupper(int c)

返回一个大写的int值

如果原来就是大写,或者字符不存在大写就不管

toascii 把任意数字转化位可见字符

其实它不是函数,他是一个宏

可能上面很多函数都是宏,只是没注意
